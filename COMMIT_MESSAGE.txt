Fix Slide 6 overflow issue with responsive scrollable design

解决 Slide 6（高速互联技术）内容溢出问题

## Problem / 问题

Slide 6 (NVLink High-Speed Interconnect) had content overflow issues where 
approximately 25% of the content was cut off and invisible to users due to 
the fixed 16:9 aspect ratio container constraint.

Slide 6（高速互联技术）存在内容溢出问题，约25%的内容被截断，
用户无法看到完整内容，这是由于固定的16:9宽高比容器限制。

## Solution / 解决方案

### 1. Enable Scrolling / 启用滚动
- Added `.slide-content-scrollable` class with overflow-y: auto
- Implemented custom scrollbar styling with brand green theme (#76B900)
- Smooth scrolling experience across all devices

### 2. Content Optimization / 内容优化
- Reduced container padding: 3rem → 2rem (-33%)
- Optimized font sizes: 17-25% reduction across headings and text
- Minimized spacing: gap reduced from 1.5rem to 1rem (-33%)
- Improved content density by ~25%

### 3. Responsive Design / 响应式设计
- Desktop (>1024px): 3-column grid, optimized spacing
- Tablet (768px-1024px): 3-column grid, reduced fonts and padding
- Mobile (<768px): Single-column layout, vertical stacking

### 4. Added Semantic Classes / 添加语义化类名
- `#slide-6` ID for targeted styling
- `.nvlink-intro`, `.stats-grid`, `.stats-card`
- `.features-grid`, `.feature-card`

## Technical Details / 技术细节

### CSS Changes / CSS 变更
- Added ~140 lines of optimized CSS rules
- Implemented 35+ specific style rules for Slide 6
- Custom scrollbar styling with `::-webkit-scrollbar`
- Responsive media queries for 3 breakpoints

### HTML Changes / HTML 变更
- Added `id="slide-6"` to section element
- Applied `slide-content-scrollable` class
- Added semantic class names to content areas
- Removed forced line break for natural text flow
