激活函数

Activation Functions

为什么需要激活函数？

向神经网络引入非线性因素。如果没有激活函数，无论多少层神经网络都只能表示线性变换，无法学习复杂的非线性关系。

常用激活函数

Sigmoid

f(x) = 1/(1+e^-x)

将输入压缩到 (0, 1) 区间

应用场景

二分类问题的输出层

ReLU

f(x) = max(0, x)

目前最常用的隐藏层激活函数

优点

计算简单、不易梯度消失

Softmax

概率分布

将输出转换为概率分布

应用场景

多分类问题

传播机制

Propagation Mechanisms

前向传播

Forward Propagation

数据从输入层经过隐藏层传递到输出层，计算出预测值

输入 → 隐藏层 → 输出 → 预测值

反向传播

Backpropagation

基于链式法则，从输出层逆向计算误差并逐层传递到输入层

计算梯度 → 更新权重 → 优化模型

损失函数

Loss Functions

损失函数用于衡量模型预测值与真实值之间的差距，是模型训练的目标函数

回归任务

均方误差 MSE

MSE = (1/n)Σ(y_i - ŷ_i)²

简单直观，梯度计算简单

对异常值敏感

分类任务

交叉熵损失

二分类：

L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]

解决梯度消失问题

与 softmax 配合高效

优化算法

Optimization Algorithms

梯度下降基本思想

通过沿着梯度下降的方向更新参数，逐步将损失函数最小化

θ_(t+1) = θ_t - η·∇J(θ_t)

梯度下降变体

SGD

Stochastic Gradient Descent

每次使用一个样本计算梯度

批量梯度下降

Batch Gradient Descent

使用全部样本计算梯度

小批量梯度下降

Mini-Batch GD

使用部分样本计算梯度

改进算法

Momentum

引入动量因子，加速更新并减少震荡

RMSprop

使用指数衰减平均值减少梯度方差

Adam

结合动量法和 RMSprop，最常用

过拟合与正则化

Overfitting & Regularization

什么是过拟合？

模型在训练数据上表现很好，但在测试数据上表现很差

训练集损失低

验证集损失高

对微小变化敏感

泛化能力差

正则化技术

L2 正则化

权重衰减 Weight Decay

L = L_loss + λΣw_i²

倾向于产生较小的权重，使模型更平滑

L1 正则化

Lasso Regularization

L = L_loss + λΣ|w_i|

产生稀疏权重，实现特征选择效果

Machine Learning Fundamentals

监督学习常见算法

线性回归

Linear Regression - 用于预测连续数值，通过拟合线性关系建立输入和输出之间的映射。

逻辑回归

Logistic Regression - 用于二分类问题，通过 Sigmoid 函数将线性输出转换为概率值。

支持向量机

Support Vector Machine (SVM) - 寻找最优超平面进行分类，可处理线性和非线性问题。

决策树

Decision Tree - 通过树形结构进行决策，易于理解和解释，可用于分类和回归。

随机森林

Random Forest - 集成多棵决策树，通过投票或平均提高预测准确性和鲁棒性。

神经网络

Neural Network - 由多层神经元组成，可以学习复杂的非线性关系，是深度学习的基础。

无监督学习常见算法

K-Means 聚类

通过迭代将数据划分为 K 个簇，每个簇由其中心点代表，最小化簇内距离。

DBSCAN 密度聚类

基于密度的聚类算法，可以发现任意形状的簇，并能识别噪声点。

主成分分析（PCA）

Principal Component Analysis - 通过线性变换将数据投影到低维空间，保留最大方差。

t-SNE 非线性降维

t-Distributed Stochastic Neighbor Embedding - 将高维数据映射到 2D/3D 空间，适合可视化。

自编码器

Autoencoder - 神经网络结构，通过编码-解码过程学习数据的压缩表示，可用于降维、去噪等。

强化学习常见算法

Q-Learning

基于值函数的强化学习算法，通过学习动作-价值函数 Q(s,a) 来选择最优动作。

SARSA

State-Action-Reward-State-Action - 在策略上（on-policy）学习，考虑实际执行的动作。

深度 Q 网络（DQN）

Deep Q Network - 结合深度学习和 Q-Learning，使用神经网络近似 Q 函数，可处理高维状态空间。

策略梯度

Policy Gradient - 直接优化策略函数，通过梯度上升最大化期望回报。

演员-评论家（Actor-Critic）

结合值函数方法和策略梯度方法，使用两个网络：演员网络输出动作，评论家网络评估动作价值。

反向传播算法详细步骤

01

计算输出层误差

将模型的输出值与目标真实值进行比较，计算误差（使用损失函数）。

02

计算梯度（链式法则）

通过链式法则，逐层向前计算每一层参数对总损失的贡献（偏导数）。这个过程从输出层开始，逐层传递到输入层。

03

更新权重和偏置

根据计算得到的梯度和学习率，使用梯度下降法更新每一层的权重和偏置参数。

w_new = w_old - learning_rate × gradient
04

重复迭代

重复进行前向传播和反向传播，直到损失函数收敛到最小值或达到预设的最大迭代次数。

防止过拟合的方法

早停法（Early Stopping）

监控验证集上的损失，当验证集损失不再改善或开始上升时，停止训练。这样可以在模型开始过拟合之前及时停止。

Dropout（随机失活）

在训练过程中随机将一部分神经元的输出设置为零，减少神经元之间的依赖关系，迫使网络学习更鲁棒的特征表示。

数据增强（Data Augmentation）

通过对训练数据进行变换（如旋转、翻转、缩放、裁剪、加噪声等）来增加数据的多样性和数量，提高模型的泛化能力。

集成学习（Ensemble Learning）

训练多个不同的模型，然后组合它们的预测结果（如投票、平均）。集成方法可以减少过拟合风险，提高整体性能。

减少模型复杂度

使用较少的层数、神经元数量或参数，降低模型的表达能力，使其更难学习训练数据中的噪声。

增加训练数据

更多的训练数据可以帮助模型学习更通用的模式，而不是记忆特定的训练样本。
